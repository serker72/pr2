Просмотреть карту позиции Вы можете, нажав на кнопку  у нужной позиции в прайс-листе или разделе "Номенклатура".

Карты оборудования и опции оборудования немного отличаются по набору доступных полей. Рассмотрим карту оборудования.

Рис. 5.14 Карта позиции.

Поля (1):

  • Раздел
  • Производитель
  • Категория
  • Подкатегория

позволяют указать местонахождение данной позиции в прайс-листе.

Поле "Форма для КП" (2) позволяет указать, какая из печатных форм должна быть использована при формировании pdf-версии КП. Используйте значение "!Единая форма!".

Поле "Текст для КП" (3) позволяет внести для оборудования полноформатный текст с картинками, используемый в pdf-версии КП.

Поле "Фото для КП" (4) позволяет загрузить в карту фото оборудования, используемое на титульной странице pdf-версии КП.
При выборе в поле (2) значения "!Единая форма!" поле (4) не используется, и фото для титульной страницы необходимо внести непосредственно в поле (3).

Блок "Добавить позицию в прайс-лист" (5) служит для добавления (удаления) позиции из прайс-листа.
Если эта галочка не отмечена, то такая позиция не занесена в прайс-лист, но доступна в разделе "Номенклатура".
Эту галочку снять невозможно, если у Вас нет соответствующих прав (на удаление позиции из прайс-листа), а также - если по позиции есть какие-либо утвержденные коммерческие предложения.

В этом блоке (5) указаны все необходимые данные для включения позиции в прайс-листы и автоматического расчета цен в прайс-листах (всех форм прайс-листа поставщика):

  • Цена (базовая цена поставщика)
  • доставка до Москвы
  • Сбор
  • СВХ, брокер
  • Базовая скидка поставщика
  • Дополнительная скидка поставщика
  • Рентабельность ExW
  • Рентабельность DDPM
  • Доставка до границы
  • Доставка после границы, руб. без НДС
  • Пошлина, %
  • Затраты брокера

Блок "Опции позиции" (6) позволяет внести в программу опции для данного оборудования и редактировать существующие опции.
Опции для оборудования создаются только путем нажатия кнопки "Добавить опцию" и заполнения карты опции, аналогичной (за исключением отдельных полей) карте оборудования.

Блок "Взаимосвязь опций" (7) позволяет задавать и редактировать правила взаимосвязи опций.

Для создания правила нажмите кнопку "Добавить правило". Откроется окно создания правила:

Рис. Окно создания правила взаимосвязи опций.

В этом окне:

  • Вид - способ взаимосвязи опций по данному правилу. Доступны следующие способы:
    1. взаимоисключающие опции - все опции по этому правилу взаимоисключающие, включение одной из них ведет к исключению остальных. например, А - главная опция, В,С - подчиненные. можно выбрать только А, В или С.
    2. главная опция вместе с обязательной подчиненной - при выборе главной опции автоматически подставится та из подчиненных опций, для которой отмечена галочка "обязательное условие". и наоборот. например, А - главная опция, В,С - подчиненные. В - "обязательное условие". Выбираем А - автоматически выбирается В.
    3. связанные опции (все одновременно) - при выборе хотя бы одной из опций по этому правилу автоматически выберутся все остальные, при снятии выбора - все опции из данного набора будут исключены. например, А - главная опция, В,С - подчиненные. при выборе хотя бы одной опции автоматически выберутся все: А+В+С
    4. связанные опции (все подчиненные) - для выбора главной опции необходим выбор всех подчиненных опций - они автоматически выбираются при выборе главной опции. Снятие хотя бы одной опции из набора ведет к снятию выбора главной опции. Но выбор подчиненных опций не ведет к выбору главной опции. например, А - главная опция, В,С - подчиненные. Выбрали А - выберется А+В+С. выбрали В или С - другие опции не выбираются.
    5. связанные опции (только одна подчиненная) - для выбора главной опции необходим выбор только одной подчиненной опции. например, А - главная опция, В,С - подчиненные. Можно выбрать следующие сочетания: А+В, А+С.
    6. связанным опциям необходима главная - при выборе любой опции из набора подчиненных опций автоматически выбирается главная опция. например, А - главная опция, В,С - подчиненные. Можно выбрать следующие сочетания: А+В, А+С, А+В+С
  • главная опция - выбор главной опции в правиле
  • ограничение по количеству - в видах взаимосвязи, где автоматически происходит совместный выбор опций, можно указать в этом поле число, отличное от нуля. В этом случае автоматически подставляется это число в количество главной опции.
  • строгое ограничение - при выборе этого поля совместно с пунктом "ограничение по количеству" - количество главной опции нельзя редактировать

После заполнения данных правила нужно нажать кнопку "Добавить правило". Правило будет добавлено в блок "Взаимосвязь опций" (7). Затем с помощью кнопки "+" в строке этого правила необходимо добавить по одной все подчиненные опции.
При нажатии "+" откроется окно создания связанной (подчиненной) опции:

Рис. Окно создания/редактирования связанной опции для правила взаимосвязи опций.

Поля этого окна аналогичны описанным выше полям. Галочка "обязательное условие" используется в виде связи "главная опция вместе с обязательной подчиненной", в остальных видах связи она не используется.

 

 

 

Вернемся в карту позиции оборудования.

Переключатель «Позиция активна» (8) при наличии у вас специальных прав позволяет устанавливать или снимать активность позиции.

В КП можно отобрать только активные позиции.

Неактивные позиции в КП добавить невозможно.

При наличии у Вас прав на редактирование будут доступны кнопки «Сохранить и перейти к списку позиций» (9) и «Сохранить и остаться» (10).

Кнопка «Закрыть форму» (11) позволяет закрыть карту без сохранения изменений.

Кнопка «Файлы…» позволяет перейти в файловый реестр этой позиции .
