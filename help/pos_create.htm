Создание оборудования

Создание позиции при наличии у Вас прав на данное действие производится с помощью кнопки «Добавить позицию…» в разделе «Прайс-лист», при выбранном виде цен "от поставщика".

При нажатии кнопки «Добавить позицию …» открывается окно создания позиции, полностью аналогичное окну редактирования:

Рис. 5.16 Окно создания позиции.

После заполнения полей нажмите кнопку «Создать позицию» для сохранения изменений и перехода в раздел «Прайс-лист» или «Создать позицию и перейти к редактированию», чтобы сохранить изменения и остаться в карте позиции.

Кнопка «Закрыть форму» позволяет закрыть карту без сохранения изменений.

 

Создание опции

Создание опции производится из карты оборудования, к которому относится эта опция. Для создания опции нажмите кнопку "Добавить опцию" в блоке "Опции оборудования". Откроется окно создания опции. Оно аналогично окну создания оборудования, за исключениями, рассмотренными ниже:

Рис. Окно создания опции.

В блоке "Группа опций" (1) необходимо выбрать группу для опции. Если требуемой группы в списке нет, вы можете создать ее в редакторе групп опций, нажав для его вызова кнопку "Группы опций".

Для опций ПНР необходимо отметить галочку "Опция установки (ПНР)" (2).

Блок "Обязательная, рекомендуемая опция" (3) позволяет указать, является ли эта опция обязательной или рекомендуемой.

Блок "Рекомендуемое количество" (4) позволяет указать рекомендуемое количество опции.

Блоки (3) и (4) следует использовать только для одиночных опций. Если эта опция является обязательной или рекомендуемой в сочетании с другими опциями, для указания такого режима работы используйте блок "Взаимосвязь опций" в карте родительского оборудования.

Блок "Добавить позицию в прайс-лист" (5) - в этом блоке есть только поле "Цена" (базовая цена поставщика). Другие поля берутся автоматически из карты родительского оборудования.

После заполнения полей нажмите кнопку «Создать позицию» для сохранения изменений и перехода в карту родительского оборудования или «Создать позицию и перейти к редактированию», чтобы сохранить изменения и остаться в карте позиции-опции.

Кнопка «Закрыть форму» позволяет закрыть карту без сохранения изменений.
