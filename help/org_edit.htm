Просмотреть карту организации Вы можете, нажав на кнопку  у нужной организации.

Откроется карта организации:

Рис. 4.3 Карта организации. Вкладка «Организация».

Карта организации состоит из двух вкладок:

  • (1) – вкладка «Организация» - сведения об организации
  • (2) – вкладка «Пользователи организации»

Рассмотрим вкладку «Организации» (1).

На этой вкладке приведены основные данные об организации.

Поля «Руководитель» и «Главный бухгалтер» (3) необходимы для утверждения карты организации.

Флаг «Карта активна» (5) делает карту активной, т.е. данную организацию можно будет выбирать для работы при входе в программу.

Внимание!!! Не снимайте данный флаг без явной необходимости такого действия. Это сделает недоступной организацию со всеми ее документами до тех пор, пока флаг не будет восстановлен!

При наличии прав на редактирование у Вас будут доступны кнопки «Сохранить и перейти к списку организаций» (6) и «Сохранить и остаться» (7).

Кнопка «Закрыть форму» (8) закрывает форму без сохранения изменений.

На Рис. 4.3 поля карты невозможно редактировать, т.к. у карты утверждено заполнение. Чтобы внести изменения в карту, необходимо сначала снять флаг «Утверждаю заполнение» (4) и нажать кнопку «Сохранить и остаться» (7). Карта примет следующий вид:

Рис. 4.4 Карта организации в режиме редактирования.

Как видно на Рис. 4.4, поля стали доступными для редактирования.

Кроме того, в карте организации (рис. 4.4) появились дополнительные кнопки у блоков:

(1) Города организации

(2)   Контактная информация

(3)   Фактические адреса

(4)   Реквизиты

 

Блок «Контактная информация» (2) представляет собой справочник контактов. Можно ввести любое количество контактных лиц с любым количеством контактов.

Ввести новое контактное лицо можно с помощью кнопки «Добавить контакты…»:

Рис. 4.8 Окно создания/редактирования контактного лица.

В этом окне нужно ввести Фамилию, имя, отчество контактного лица и его должность и нажать кнопку «Добавить».

Контактное лицо можно редактировать. Для этого служит кнопка  под данными контактного лица. Откроется окно, аналогичное Рис. 4.8.

Добавить контакт контактному лицу возможно при помощи кнопки  в колонке «Контакты» в строке данного контактного лица. Откроется окно создания/редактирования контакта:

Рис. 4.9 Окно создания/редактирования контакта.

Необходимо выбрать вид контакта и вписать его значение, затем нажать кнопку «Добавить». Введенные контакты можно редактировать с помощью кнопки  справа от соответствующего контакта. При этом откроется окно, аналогичное Рис. 4.9

При наличии прав возможно удаление контактов.

Для удаления контактного лица со всеми его контактами используйте кнопку  в крайней правой колонке справочника контактов.

Для удаления только одного контакта используйте кнопку  справа от этого контакта.

Блок «Фактические адреса» (3) на Рис. 4.4.

В этом блоке вводятся фактические адреса организации. Этих адресов может быть несколько.

Для создания нового адреса нажмите кнопку «Добавить фактические адреса…». Откроется окно создания/редактирования фактического адреса:

Рис. 4.10 Окно создания/редактирования фактического адреса.

Необходимо вписать адрес и нажать кнопку «Добавить».

Адреса можно редактировать при помощи кнопки  справа от редактируемого адреса.

Откроется окно, аналогичное Рис. 4.10.

При наличии прав возможно удаление адресов.

Для удаления адреса  нажмите кнопку  справа от этого адреса.

Блок «Реквизиты» (4) на Рис. 4.4 позволяет задавать реквизиты организации.

Для создания новых реквизитов нажмите кнопку «Добавить реквизиты…».

Откроется окно создания/редактирования реквизитов:

Рис. 4.11 Окно создания/редактирования реквизитов.

Необходимо заполнить поля окна и нажать кнопку «Добавить».

Флаг «Основные реквизиты» позволяет назначить вводимые реквизиты основными для этой организации. Основные реквизиты подставляются по умолчанию в качестве реквизитов организации при создании различных документов в программе.

Реквизиты можно редактировать при помощи кнопки  справа от редактируемых реквизитов.

Откроется окно, аналогичное Рис. 4.11.

При наличии прав возможно удаление реквизитов.

Для удаления реквизитов  нажмите кнопку  справа от удаляемых реквизитов.

Кнопка «Учредительные документы» в карте организации (Рис. 4.4) открывает файловый реестр, содержащий учредительные документы:

Рис. 4.12 Фрагмент окна реестра учредительных документов организации.

Раздел представляет собой реестр доступных для скачивания файлов (2). В реестре указаны:

  • имена файлов,
  • размер в мегабайтах,
  • кто из сотрудников загрузил файл,
  • описание каждого файла,
  • даты загрузки.

Чтобы скачать файл, необходимо либо кликнуть по имени файла, либо нажать кнопку Получить.

В зависимости от Ваших прав, у Вас могут быть показаны блоки и кнопки:

(1)   Загрузить файл. С помощью этого блока Вы можете загрузить файл в реестр. Необходимо описать файл в поле «Описание файла», и затем нажать кнопку «Выберите и загрузите файл…». Откроется окно выбора файла. После выбора файл будет загружен в раздел.

(2)   В блоке (2) – кнопки:

   - редактировать описание файла;

 - удалить файл.

Вкладка «Пользователи организации» (2) на Рис. 4.3.

На данной вкладке показан список сотрудников, и отмечены те сотрудники, которые имеют доступ в программе к данной организации:

Рис. 4.13 Вкладка «Пользователи организации».

Внимание! Для редактирования списка сотрудников необходимо снять утверждение заполнение карты организации и сохранить изменения.

Красным выделены сотрудники, карты которых неактивны.

После отметки нужных сотрудников нажмите кнопку (1) «Сохранить и перейти к списку организаций», чтобы перейти в реестр организаций, или кнопку (2) «Сохранить и остаться», чтобы остаться в карте организации.

Кнопка «Закрыть форму» (3) позволяет выйти из карты, не сохраняя изменения.
