Полное наименование ОПФ ИНН КПП Город Контакт Карта
  • Поиск контрагентов возможен по нескольким критериям в каждом поле.
  • Для этого укажите требуемые значения через точку с запятой.
  • Например, поиск по критерию:  гласс;первый   в поле "Полное наименование" позволит найти всех контрагентов, у которых в названии встречаются сочетания букв "гласс", либо "первый".
