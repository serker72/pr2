Период








%{if $can_print}% %{else}% %{/if}%




1. В отчет попадают все утвержденные исходящие счета в статусе "Выполнен", по позициям которых +/- к выдаче совпадает с полученным +/-, либо +/- получен частично.
Если Вы не выровняли позиции в исходящем счете, то он не попадает в отчет.

2. Фильтр "Период". По умолчанию отчет формируется изо всех документов по п.1 за все время работы программы. При выборе периода и нажатии кнопки программа отбирает исходящие счета по датам выдачи +/-.
3. Фильтр "Контрагент". Вы можете осуществлять отбор исходящих счетов по нескольким контрагентам.
Для этого в форму поиска нужно вписать фрагмент названия искомого контрагента. Программа выдаст список вариантов названий. Выберите в списке требуемого контрагента щелчком мыши по его названию. При необходимости отобрать в отчет счета по еще одному контрагенту - снова наберите фрагмент его названия и выберите контрагента из всплывшего списка вариантов названий контрагентов.
После выбора всех необходимых контрагентов нажмите кнопку .
