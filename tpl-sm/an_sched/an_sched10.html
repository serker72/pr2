Фильтр Период: поиск данных осуществляется следующим образом:
Звонки - наличие хотя бы одного звонка по каждому контрагенту, дата утверждения выполнения попадает в выбранный период;
Встречи - наличие хотя бы одной встречи по активности "Встречи" планировщика + хотя бы одной встречи по активности "Командировки" планировщика, для которой не отмечена галочка "не встречался", по каждому контрагенту, дата утверждения выполнения попадает в выбранный период;
КП - дата утверждения коммерческого предложения попадает в выбранный период;
Лиды - во всех статусах, кроме создан, аннулирован, дата утверждения заполнения лида попадает в выбранный период;
Контракты - строится по двум видам документов:
Тендеры в статусе выигран, дата утверждения выполнения попадает в выбранный период; Договоры и приложения - месяц, год договора попадает в выбранный период.
